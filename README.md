# FitVault Full Starter

A fuller starter repo for your fitness vault app.

## Includes
- Home dashboard
- Gym check-in
- Session logging
- Saved foods vault
- Nutrition logging
- Progress tracking
- Media vault
- Settings with theme switching
- Cloudflare Worker API
- D1 schema
- R2 upload support

## Setup
1. Upload repo to GitHub
2. Connect to Cloudflare Workers/Pages
3. Run the D1 schema from `schema.sql`
4. Deploy

## Notes
- This build avoids paid AI APIs for now
- OCR endpoint is a stub-ready route for later
- Media display uses your public R2 URL already wired in
